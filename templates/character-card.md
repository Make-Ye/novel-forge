# Character Card: <!-- Character Name -->

> Three-layer character profile. Surface is what the world sees. Inner is what the character hides. Core is what drives everything.
>
> **增量成长**: Phase 4 填写 Identity + Surface Layer + Core Layer（核心创伤/核心信念）即可。Inner Layer 和 Character Arc 在写作中随角色展现逐步补充。Phase 8 角色做出意外之事时捕获到文件。**不要**在 Phase 4 预填所有细节。
>
> **层级**: Tier 1 主角用完整模板（characters/main/）。Tier 2 配角用简化版（characters/supporting/）：仅需 Identity + Surface Layer 概要 + Key Relationships + 叙事功能。出场≥3次的配角升级为 Tier 1。
